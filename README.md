# Yubin Kim Research Portfolio

GitHub Pages 또는 일반 정적 웹 호스팅에 바로 올릴 수 있는 개인 연구 홈페이지입니다.

## 파일 구조

- `index.html`: 홈페이지 본문
- `styles.css`: 디자인 및 반응형 배치
- `script.js`: 한글/영문 전환, 다크 모드, 발표 목록 확장
- `assets/Yubin_Kim_CV_KR.pdf`: 국문 이력서
- `assets/yubin-kim-profile.jpg`: 프로필 사진
- `assets/axis_3d_plot.html`: 연애·결혼 담론 연구의 Interactive 3D Visualization 예시
- `assets/jobplanet-wlb-example.png`: 잡플래닛 연구 결과 시각화 예시

## GitHub Pages 게시

1. 새 저장소를 만들거나 기존 저장소를 엽니다.
2. 압축을 해제한 뒤, `index.html`이 저장소의 최상위 경로에 오도록 모든 파일과 `assets` 폴더를 업로드합니다.
3. 저장소의 `Settings` → `Pages`로 이동합니다.
4. `Deploy from a branch`를 선택하고 `main` 브랜치의 `/ (root)`를 지정합니다.
5. 저장 후 안내되는 주소로 접속합니다.

## 수정 시 주의

- 연구 이미지에는 탐색적 분석 예시 또는 연구 자료 예시라는 설명이 포함되어 있습니다.
- LinkedIn 주소와 이메일은 `index.html`에서 수정할 수 있습니다.
- 영문과 한글 문구는 각각 `lang-en`, `lang-ko` 클래스로 구분되어 있습니다.

## 이번 수정 사항

- 상단 연구 분야를 사회심리학과 계산사회과학으로 수정했습니다.
